# ExamGuard Icons

Thư mục này chứa icon tĩnh cho TeacherForm và StudentForm.

Workflow:

1. Chọn icon từ IconArchive hoặc nguồn có license rõ ràng.
2. Lưu icon vào repo, ví dụ `desktop/assets/icons/connect.png`.
3. Cập nhật `ATTRIBUTION.md`.
4. Gắn icon vào WinForms ở build-time, không tải asset runtime.

Các nhóm icon nên có:

- kết nối / ngắt kết nối
- nộp bài
- chat
- giơ tay
- cảnh báo
- khóa / mở khóa
- phóng to / đóng
