# Icon Attribution

ExamGuard dùng icon dưới dạng asset tĩnh trong repo, không tải icon từ Internet khi ứng dụng chạy.

Nguồn đề xuất: https://www.iconarchive.com/

Quy tắc khi thêm icon:

- Chỉ dùng pack có giấy phép phù hợp, ưu tiên mục `Commercial free Packs`.
- Tải icon về `desktop/assets/icons/` dưới dạng PNG/ICO/SVG.
- Ghi rõ tên pack, tác giả, URL, license và ngày tải trong file này.
- Không hotlink ảnh trực tiếp từ IconArchive hoặc nguồn bên ngoài.
- Không commit icon có license chỉ cho mục đích cá nhân nếu bản build dùng trong trường học/doanh nghiệp.
