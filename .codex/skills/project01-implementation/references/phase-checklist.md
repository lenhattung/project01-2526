# Phase Checklist

## Phase 2 Baseline

- Teacher chat, attention, lock/unlock, command, file distribution, teacher broadcast
- Student receive or execute display behavior
- Remote mouse click from Teacher detail view
- Docker Compose backend and MySQL

## Phase 3 Baseline

- Teacher LAN auto-discovery broadcast
- Student discover teacher
- Remote text input
- Remote clipboard set
- Protocol smoke tests
- Desktop publish script

## Good Next Steps

- Full remote-control keyboard shortcuts, drag, and permission prompts
- Backend-driven teacher session or class selection instead of manual IDs
- Stricter website blocking through Windows policy, DNS, firewall, or hosts strategy
- Student background service or startup agent
- Reporting and audit dashboard
- Signed release packaging and installer flow
