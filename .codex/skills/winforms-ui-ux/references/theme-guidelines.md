# Theme Guidelines

## Visual Direction

- Use a calm, professional classroom-operations look.
- Prefer light surfaces with strong contrast and a restrained teal accent.
- Keep panels and tool surfaces distinct with subtle borders instead of heavy 3D chrome.

## Control Rules

- Buttons: flat, filled for primary, neutral for secondary, consistent height and padding.
- Inputs: white surface, subtle border, dark text, no oversized heights.
- Lists and logs: high-contrast text, visible selected state, no cramped columns.
- Status chips and labels: short, readable, use color as support rather than sole signal.

## Layout Rules

- Group connection or session controls together.
- Group high-risk control actions together and make target scope obvious.
- Keep image panels dark so screen frames remain readable.
- Keep the most recent log entries immediately visible without extra clicks.
